# Healthcare Analytics Dashboard

## Overview
Healthcare Analytics & Decision Support Dashboard developed using Excel, MySQL, Python, and Power BI.

## Key Insights

- Orthopedics generated the highest revenue.
- Back Pain is the most common disease.
- 51.3% of patients have insurance coverage.
- Average length of stay is 7.96 days.
- Total revenue generated is 76M.

## Tools Used
- Excel
- MySQL
- SQL
- Python
- Pandas
- Matplotlib
- Power BI

## Features
- Revenue Analysis
- Disease Analysis
- Insurance Coverage Analysis
- Patient Status Analysis
- Length of Stay Analysis
- Interactive Dashboard
